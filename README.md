# React 19 Playground

A playground project to explore and demonstrate the new features in React 19 with TypeScript and styled-components.

## Getting Started

### Prerequisites
- Node.js (version 16 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd what-is-new-in-react-19
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Project

#### Development Mode
Start the development server:
```bash
npm run dev
```
The application will be available at `http://localhost:3000`. Note: The browser will not open automatically - navigate to the URL manually.

#### Production Build
Build the project for production:
```bash
npm run build
```

#### Type Checking
Run TypeScript type checking:
```bash
npm run type-check
```

#### Linting
Check code quality:
```bash
npm run lint
```

Fix linting issues automatically:
```bash
npm run lint:fix
```

## Project Structure
- `src/` - Source code
- `public/` - Static assets
- `dist/` - Production build output

## Technologies Used
- React 19 (latest)
- TypeScript
- styled-components
- Webpack 5
- ESLint
